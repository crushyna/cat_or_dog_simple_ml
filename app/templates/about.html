Simple demo of web application that can recognize animal (cat or dog) from a selected photography.

Created using Flask UWSGI Docker image (by tiangolo) and machine learning server (code available here).

How it works:

  • User sends image to webapp (<5MB)
  • Input image is resized twice:
    • For result page (400x250px max)
    • For ML engine (160x160px)
  • Smaller image (160x160) is sent to ML Server (Azure Function App)
  • Function App recognizes animal (cat or dog) from the image, using pre-trained Tensorflow model
  • Function App returns information, whether it is a cat, dog or unknown creature (when score is lower than 80%)

Pros:

  • it is cheap! (due to sleepy nature of Function App)
  • most of the time it works! (thanks to me :) )

Cons:

  • Function App actually can take some time to start after long time of sleep
  • There's no image padding implemented, so make sure that animal covers most of the photography

...and as you can see, front-end is my passion :)
